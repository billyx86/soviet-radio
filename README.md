# Радио Маяк — Soviet Radio

A bakelite-and-amber Soviet-era shortwave radio web app with real internet radio streams.

## Stack
- React 19 + TypeScript
- Vite + TanStack Start
- Tailwind CSS v4
- HTML5 Audio

## Run
```bash
npm install
npm run dev   # 0.0.0.0:8080
```

## Features
- Physical knobs (volume, tuning)
- Power switch, band selector
- Amber dial, signal meter, speaker grille
- Cyrillic/English panel labels
- Working HTTPS streams (SomaFM etc.) presented as shortwave channels
